# 🌙 NuitkaGUI

> A Professional GUI Frontend for Nuitka\
> Make Python Packaging Visual, Stable and Simple.

------------------------------------------------------------------------

## 🚀 什么是 NuitkaGUI？

NuitkaGUI 是一个基于 PySide6 构建的 **Nuitka 可视化打包工具**。

它将原本复杂的命令行参数转换为直观的图形界面操作，让开发者能够：

-   拖拽 `.py` 文件完成打包
-   一键选择图标
-   可视化配置打包模式
-   自动检测虚拟环境
-   自动创建输出目录
-   安全稳定地执行打包流程

------------------------------------------------------------------------

## ✨ 核心特性

### 📦 一键打包

拖入 Python 文件 → 自动生成输出目录 → 选择模式 → 开始打包。

无需手写命令。

------------------------------------------------------------------------

### 🧠 智能环境识别

-   自动检测项目内 `python.exe`
-   支持虚拟环境优先
-   自动同步 Model / View 状态

------------------------------------------------------------------------

### 🛡 工业级稳定性

-   防止 NoneType 崩溃
-   AST 解析异常自动跳过
-   线程安全封装
-   统一异常处理
-   输出目录安全创建

------------------------------------------------------------------------

### ⚙ 支持打包模式

  standalone独立目录模式   onefile单文件模式

支持 Windows Console Mode：

-   force
-   disable
-   attach

## 🛠 从源码运行

``` bash
git clone https://github.com/HeiKe-Tom/NuitkaGUI.git
cd NuitkaGUI
pip install -r requirements.txt
python NuitkaGUI.py
```

## 🔧 技术架构

-   UI 框架：PySide6
-   架构模式：MVP
-   编译工具：Nuitka
-   日志系统：loguru
-   线程封装：RunInThread
-   单例管理：singleton 装饰器
